Smart Health Tracker

Overview

Smart Health Tracker is a Java-based application developed to help users monitor and manage their daily health activities.

Features:

* User Authentication
* Activity Tracking
* Diet Tracking
* Sleep Monitoring
* Water Intake Tracking
* Health Summary Reports

Technologies Used:

* Java
* Object-Oriented Programming (OOP)
* NetBeans IDE

OOP Concepts Applied:

* Encapsulation
* Inheritance
* Polymorphism
